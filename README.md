# Fullstack Auth + Book App

A fullstack web application built with **Angular 20** and **C# .NET 9 Web API**. It includes user authentication using **JWT** and a simple book management system.

---

## Features

- User registration and login
- JWT authentication
- Protected routes
- Add and view books and citations
- SQLite database

---

## Tech Stack

- Angular 20
- .NET 9 Web API
- SQLite
- JWT Authentication

---

## Getting Started

### Prerequisites

Make sure you have installed:

- Node.js
- Angular CLI
- .NET SDK 9
